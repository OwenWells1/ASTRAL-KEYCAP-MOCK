Title:

Astral Keycap Mock



Description:

A quick CAD mock-up of a keycap with decorative fans attached. Built as an exercise to show use of assemblies, joints and real world translation in fusion 360. 



Usage:

Download .f3z file and open into fusion 360.
Move fans with mouse left click to view joints functions.
Select Joints folder, right click, choose animate joint to view simulated motion.
OR 
Upload .obj file into Autodesk viewer to use within browser.
This is visual only, no joints.



Design intent:

Familiarise myself with fusion360 again after time away from it.
This design was chosen as a simple choice with various useful features to remember forgotten features.
